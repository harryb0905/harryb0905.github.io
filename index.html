Info:

I am a 20 year old developer with a passion for Computing. I'm currently studying Computer Science (MSci) at Lancaster University. I chose Computing as I enjoy problem solving, designing and implementing algorithms and learning about computer architecture. Logical problem solving has always appealed to me and I find the mathematical aspects of Computing to be of great interest to me.

Skills:

During my degree I have been exposed to many languages. Java was used to develop our understanding of Object Orientation, and Python to enhance our Dynamic Scripting abilities. Other languages I have studied include; C, PHP, Erlang and Processing.

In my free time I have learned several web languages. I am proficient in HTML, CSS and JavaScript. I am interested in many JavaScript frameworks, such as jQuery, Node, Electron and React. A recent JavaScript project I developed was a desktop-based task manager application, using React, Electron and Firebase (see my portfolio for more details).

I am very confident with the major Object Oriented principles. In the Advanced Programming module in my degree, we were asked to create an OO representation of the Solar System in Java.

I use Git for Version Control whenever I am programming. I find it a very useful tool for collaboration on team projects and for managing changes to source code over time in my own personal projects.

In Second Year of University in the Operating Systems module, we were asked to implement a Java program that could traverse an ext2 filesystem partition. This was incredibly fascinating learning about how filesystems are represented in computers. We were also introduced to key concurrency concepts such as multi-threading, locks and semaphores which help to provide faster execution times than sequential code.

I am currently learning the Swift language which will enable me to publish apps to the iOS app store in the near future.
