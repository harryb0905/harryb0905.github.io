About Me

My name is Harry Baines, I'm 20 years old and I live in Lancashire, UK. I'm currently studying Computer Science (MSci) at Lancaster University. I chose Computing as I enjoy problem solving, designing and implementing algorithms and learning about computer architecture. Logical problem solving has always appealed to me and I find the mathematical aspects of Computing to be of great interest to me. As a developer I thoroughly enjoy developing exciting and innovative applications. I'm proficient in web languages, such as HTML, CSS and JavaScript. I am interested in many JavaScript frameworks, such as React, Node, and Electron (see my portfolio). I also enjoy working with Python and Java. In my free time I enjoy running and playing tennis.

Description:

Having fish in your own garden will require constant monitoring and great care. I decided to create a wireless livestream using a Raspberry Pi and the camera module, which would allow any user in our house to view the livestream 24/7 on any device connected to the internet. The aquatic life in the pond can be viewed via the website I created on our local network.

Blink3

Description:

I teamed up with Nathaniel to create a desktop client application for the Blinkstick. The Blinkstick is a very basic programmable LED device. We decided to create an all-in-one application which would enable the user to change the color, brightness and various other features of their Blinkstick once plugged into their computer.
